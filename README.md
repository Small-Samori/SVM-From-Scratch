# Overview
In this project, an SVM algorithm was built from scratch using numpy.
The algorithm was then made to classify randomly generated data.


# Dependencies
* Numpy
* Matplotlib

# Usage
1. `cd '\SVM From Scratch' `
1. `python 'SVM algorithm.py'`
